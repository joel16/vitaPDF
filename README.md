# vitaPDF ![Github latest downloads](https://img.shields.io/github/downloads/joel16/vitaPDF/total.svg)

A simple homebrew file browser that is used for viewing various documents on the PlayStation VITA.

# Supported Formats:
- CBT
- CBZ
- EPUB
- FB2
- MOBI
- PDF
- XPS
